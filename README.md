# Privanote - Simple note application

**Privanote** is a simple, self-hosted note-taking app with:
- a web-based GUI (React + TailwindCSS),
- a local REST API (Django),
- and SQLite for data storage.

## Technologies used

- **Docker**
- **uv**
- **npm**

- **Django**
- **React**
- **TailwindCSS**
